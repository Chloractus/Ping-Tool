# Scapy-Tools
Multiple Python Cybersecurity scripts for various uses.

These Python scripts center around the use of Scapy (A Python library that allows for the Sending, Recieveing, and Crafting of Packets). All of these scripts are meant to help others learn about how Python scripts can be used to harden or exploit systems. That being said, these scripts require a medium to high level of Python knowledge and are not recommended for beginners in Python and/or Cybersecurity.
